# シューティングゲーム仕様書 - コアメカニクス

本ドキュメントは、全6ステージ構成のシューティングゲームにおけるコアメカニクス（基本システム）の仕様を定義します。

---

## 1. 基本概要
* **ステージ数**: 全6ステージ
* **ゲームジャンル**: 縦長スクロール（1画面に収まる範囲）
* **画面アスペクト比**: 縦長比 4:3
* **対応プラットフォーム**: PCおよびスマートフォン想定

---

## 2. プレイヤー（自機）仕様

### 2.1 移動仕様
* **PC操作**: WASDキーによる上下左右移動。
* **スマートフォン操作**: 画面右下に配置されたバーチャル十字キーによる移動。
* **移動速度**:
  * **低速移動（ショット発射中）**: 弾幕回避用の基本速度
  * **高速移動（ショット非発射中）**: 低速移動時の1.5倍の速度

### 2.2 当たり判定
* プレイヤー側の当たり判定は極めて小さく、敵の「弾幕」と同じぐらいのコンパクトなサイズに設計。

### 2.3 ショット（通常攻撃）仕様
* **発動方法**: 
  * **PC**: スペースキーの長押し
  * **スマートフォン**: 画面右下に配置されたショット操作エリア（ボタン）の長押し
* **発射頻度**: 0.5秒ごとに1発

### 2.4 ボム（緊急回避）仕様
* **効果**: 自機周辺（画面全体の4分の1程度の広さ）にある敵の弾幕のみを消去する。
* **初期所持数**: 5個
* **発動方法**: 
  * **PC**: Shiftキーの押下
  * **スマートフォン**: ボムアイコンの直接タップ
* **ボムのドロップ（補充）条件**:
  * 各ステージのボスに挑む直前に1個ドロップ
  * ボスを撃破した際に2個ドロップ

### 2.5 残機（ライフ）と復活仕様
* **初期残機**: 0（ステージ開始時点）
* **残機増加方法**: 1つのステージにおいて、出現する雑魚敵10体をすべて倒した場合のみ、ドロップ品として「残機」アイテムが1つ出現。獲得することで残機が+1される。
* **持ち越し**: 獲得した残機は、次のステージ以降にそのまま引き継いで使用可能。
* **被弾時の復活**: 残機が1以上ある状態で被弾した場合、残機を1消費し、その場で**4.5秒間無敵状態になって復活**する。

### 2.6 アイテム（ドロップ品）仕様
* **共通挙動**: ボムアイテムや残機アイテムなどのドロップ品は、出現箇所から**下に向かってゆっくり落ちていき、自機が接触した時点で獲得**となる。

---

## 3. 敵（エネミー）仕様

### 3.1 構成と出現パターン
各ステージ（全6ステージ共通）の敵構成は以下のように固定されます。
1. **雑魚敵**: 10体（1体ずつの単独出現、および複数体の同時出現パターンの両方を想定）
2. **ボス**: 1体（雑魚敵10体の出現・処理後に登場）

### 3.2 攻撃（弾幕）パターン
* **雑魚敵**: 「自機狙い弾」または「全方位弾」のいずれか一方を使用。
* **ボス**: 「自機狙い弾」「直進弾」「全方位弾」の3種類すべてを使用。

### 3.3 敵の体力（HP）スケーリング
ステージが1つ進むごとに、敵の耐久力が以下のように段階的に上昇します。

| ステージ | 雑魚敵の体力（ショット必要ヒット数） | ボスの体力（ショット必要ヒット数） |
| :--- | :---: | :---: |
| **ステージ1** | 5発 | 100発 |
| **ステージ2** | 6発 | 110発 |
| **ステージ3** | 7発 | 120発 |
| **ステージ4** | 8発 | 130発 |
| **ステージ5** | 9発 | 140発 |
| **ステージ6** | 10発 | 150発 |

---

## 4. スコア（点数）仕様
* **雑魚敵撃破**: +5,000点
* **ボス撃破**: +100,000点

---

## 5. ゲームオーバー条件
* 残機が0の状態で被弾し、コンティニューを実行しない（またはコンティニュー可能回数を使い切った）場合にGAME OVERとなる（※画面遷移の詳細は `game_flow.md` 参照）。
